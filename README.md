# Web Absensi SMAN 1 Megamendung

Sistem absensi berbasis web menggunakan PHP dan MySQL.

## Fitur
- Login Admin
- Login Siswa
- Dashboard
- Absensi Kamera
- Data Siswa
- Laporan Absensi
- Export PDF

## Teknologi
- PHP Native
- MySQL
- HTML CSS JavaScript
- Bootstrap / Tailwind

## Cara Menjalankan
1. Simpan folder di htdocs
2. Import database di phpMyAdmin
3. Jalankan XAMPP Apache & MySQL
4. Buka localhost/web-absensi-sman1m
